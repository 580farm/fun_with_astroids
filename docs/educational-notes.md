# Educational Notes

Fun with Asteroids is both a game and a learning project. Every feature should
be teachable in small pieces.

## Python Concepts

| Game Feature | Programming Concept |
| --- | --- |
| Game settings | Constants and dataclasses |
| Player movement | Variables and vectors |
| Input handling | Conditionals and events |
| Game loop | Loops and timing |
| Ship and asteroid objects | Classes and methods |
| Collision detection | Geometry and distance |
| Score calculation | Functions and tests |
| Unlock data | Lists, dictionaries, and structured data |
| Save files | File input and output |

## Math Concepts

- Angles and rotation.
- Velocity and acceleration.
- Distance between two points.
- Random numbers.
- Timers and rates.
- Coordinate systems.

## Space History Concepts

- Differences between capsules, orbiters, cargo vehicles, and launch vehicles.
- Space Race era vehicles compared with modern spacecraft.
- Why spacecraft shapes differ by mission.
- How engineering tradeoffs affect size, durability, and maneuverability.

## Suggested Learning Rhythm

1. Play the current version.
2. Pick one tiny improvement.
3. Read the code that controls it.
4. Change one value or one function.
5. Run the game again.
6. Talk about what changed.
7. Commit the working step.

## Family-Friendly Tasks

- Rename a level.
- Change asteroid colors.
- Tune ship speed.
- Add a new spacecraft fact.
- Draw a placeholder sprite.
- Add a powerup idea.
- Write a test for score math.

## Vocabulary

- Sprite: a game object that can be drawn on screen.
- Frame: one drawn moment of the game.
- FPS: frames per second.
- Velocity: speed and direction.
- Collision: two objects touching or overlapping.
- Refactor: improve code structure without changing behavior.
